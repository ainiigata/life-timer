# 人生タイマー v5「きろく」設計 — メモ・カレンダー・コンテキスト書き出し

日付: 2026-07-13

## 目的

思いついたことをすぐ忘れてしまい、隙間時間に「何をしたかったか」を思い出せない——という課題を解決する。日々の思考をメモとして蓄積し、カレンダーで「いつどんなことを考えていたか」を振り返り、蓄積したコンテキストをAI(Claude等)に渡せるMarkdownで書き出せるようにする。

## 決定事項(ユーザー確認済み)

- 新しい5つ目のタブ**「きろく」**にメモ入力+カレンダー+エクスポートをまとめる
- カレンダーにはメモ以外の既存記録も**統合表示**(今日の問いの答え・今日の宣言・ガチャのクリアお題・叶えた夢)
- エクスポートは**Markdownで全記録**(プロフィール・優先順位・やりたいこと含む)。クリップボードコピーと.mdダウンロードの両方
- メモは**種類タグ付き**: アイデア(idea)/ やること(todo)/ メモ(memo)

## データモデル(version 1 のまま追加)

```js
notes: [],         // { id, date: 'YYYY-MM-DD', time: 'HH:MM', type: 'idea'|'todo'|'memo', text }
declarations: [],  // { date: 'YYYY-MM-DD', text, done }  過去の「今日の宣言」のアーカイブ
```

- 検証は reflections と同方式(null許容、配列なら各要素検証)。text は空不可・500文字以内
- `init()` でバックフィル(`!data.notes → []` 等)
- **宣言のアーカイブ**: `data.today.date` が過去日になったら `declarations` に移して `data.today = null`(init時とtickの日付変化時)。これで宣言の履歴がカレンダーに残る

## 新モジュール src/records.js(グローバル `Records`、DOM非依存・純関数)

| 関数 | 役割 |
|---|---|
| `monthMatrix(year, month)` | 月カレンダー用の週×7の2次元配列(日曜始まり、月外はnull) |
| `dayEntries(data, dateStr)` | その日の全記録を統合(notes時刻順 → 問いの答え → 宣言 → お題クリア → 叶えた夢)。`{kind, ...}` 形式 |
| `datesWithEntries(data, year, month)` | 記録がある日付のSet(ドット表示用) |
| `exportMarkdown(data, questionsList, now)` | プロフィール/優先順位/やりたいこと/日々のきろく(日付昇順)のMarkdown生成 |

## UI

タブバー: わたし・**きろく**・見つめる・家族・やりたいこと(素早くメモに届くよう2番目)。

`#screen-record` の構成(上から):

1. **クイックメモ** — 種類チップ(💡アイデア/✅やること/📝メモ、デフォルト=メモ)+テキスト入力+保存
2. **最近のメモ** — フィルタチップ(すべて/種類別)+新しい順に最大30件。種類バッジ+本文+日付+削除(confirm)
3. **カレンダー** — 年月ナビ+日曜始まりグリッド。記録のある日にドット、今日をハイライト。日タップで直下に詳細パネル(その日の全記録、メモは削除可)。初期選択は今日
4. **きろくを書き出す** — 「Markdownをコピー」(clipboard API、失敗時メッセージ)+「.mdをダウンロード」(Blob、`life-timer-notes-YYYY-MM-DD.md`)

描画は `renderRecordScreen()` を `render()` から呼ぶ(tickの毎秒再描画には入れない=入力保護。family/wishesと同じ扱い)。カレンダーの表示年月・選択日・フィルタはapp.jsのクロージャ変数。

## エクスポートMarkdown形式

```markdown
# 人生タイマー きろく(2026-07-13 書き出し)

## プロフィール
- 誕生日 / 性別 / 目標寿命 / 時間の優先順位

## やりたいこと
- [ ] 家族で沖縄に行く(50歳までに)★今の夢
- [x] ○○(2026-07-01 達成)

## 日々のきろく(日付昇順)
### 2026-07-13(月)
- 09:12【アイデア】…
- 【問いの答え】Q: … →「…」
- 【今日の宣言】… ✅
- 【お題クリア】(R) …
```

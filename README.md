# 人生タイマー

残り時間を可視化して、やりたいことを今やるためのアプリ。

## 特徴

- **データはローカルのみ**: すべてのデータはブラウザの `localStorage` に保存され、外部サーバーには送信されません
- **オフライン対応**: Service Worker により、ネットワークがなくても動作します
- **ファイル直開き対応**: `dist/index.html` をブラウザで直接開いて使用できます

## 使い方

### ブラウザで開く

1. `dist/index.html` をウェブブラウザで開く
2. 誕生日と性別を入力して「わたし」を登録
3. 家族を追加して、会う頻度を設定
4. やりたいことを記録して、進捗を見守る

### GitHub Pages で公開

本プロジェクトを GitHub リポジトリとして公開し、GitHub Pages を有効にすることで、以下から アクセス可能:

```
https://<username>.github.io/<repository>/
```

## ビルド

ソースコード(`src/`)を単一の HTML ファイル(`dist/index.html`)にビルド:

```bash
node build.js
```

## テスト

```bash
node --test
```

## データ管理

### バックアップ

アプリの設定画面から JSON ファイルとしてデータをエクスポートできます:

1. 設定アイコンをタップ
2. 「エクスポート」を選択
3. JSON ファイルがダウンロードされます

### リストア

エクスポートしたデータを読み込み:

1. 設定アイコンをタップ
2. 「インポート」で JSON ファイルを選択
3. 確認後、データが置き換わります

### 破損復旧

データが読み込めなくなった場合:

1. 設定アイコン下の「ファイルから復元」を選択
2. バックアップ済みの JSON ファイルを選択
3. アプリが再起動して復旧します

## 出典
